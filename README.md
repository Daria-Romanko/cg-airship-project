# Air Delivery Project

Учебный проект: небольшая 3D-игра про доставку посылок. Игрок управляет кораблём с Сантой, летит над полем и сбрасывает посылки по целям.

![Gameplay](AirshipsProject/models/screenshot.png)

## Что реализовано

### Сцена и окружение
- Поле с новогодней ёлкой (модели и текстуры для поля/ёлки/корабля).
- Глобальное освещение: направленный источник света.
- На земле размещаются **цели-домики** и **декорации**:
  - минимум 5 домиков-целей
  - минимум 1 вид домика
  - минимум 2 вида декораций
  - расположение генерируется случайным образом

### Небо и погодные эффекты
- В небе размещаются **грозовые тучки** и **воздушные шары**.
- Тучи **двигаются по траектории** (на основе тригонометрических функций).
- Тучи **периодически сверкают** за счёт сияющего материала/эффекта свечения.

### Шейдеры и графические эффекты
- **Колыхание объектов** через вершинный шейдер.
- **Normal map** для корабля.

### Камера
- Камера по умолчанию следует за кораблём (вид сзади-сверху).
- Есть переключение в режим прицеливания: камера переносится в нижнюю часть корабля.

## Управление (пример)
- `W/A/S/D` — движение по горизонтали
- `C` — переключить камеру (третье лицо / прицеливание)
- `Space` — сбросить посылку
- `Esc` — выход

## Используемые технологии
- **OpenGL**
- **SFML**
- **GLM**
- **ImGui**
- **Assimp**
